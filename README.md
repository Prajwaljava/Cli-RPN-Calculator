# Cli-RPN-Calculator
CLI RPN Calculator
# Environment
Java
## Supported Operations

|Operation	|Distraction|
|-----------|------------ |
|+	|Addition|
|-	|Subtraction|
|*	|Multiplication|
|/|	Division|
|%	|Residue|
|^	|Power|

## Usage


*   __output__

        10 

    Prints 10, the value at the top of the stack.

*   __addition__

        10 2 + 

    Prints 12, the sum of 10 and 2

*   __subtraction__

        100 50 - 

    Prints 50

*   __division__

        12 4 / 

    Prints 3

*   __multiplication__

        3 3 * 

    Prints 9
